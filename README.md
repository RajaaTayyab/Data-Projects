# Netflix Data Exploration

This project performs an **exploratory data analysis (EDA)** on a Netflix dataset using **Python**, **Pandas**, and **Matplotlib**.  

The dataset was cleaned by:  
- Handling missing values in the `director` and `country` columns.  
- Converting date strings into **datetime objects**.  

**Key findings** include:  
- Movies dominate TV Shows in the dataset.  
- A steady increase in content releases starting from **2015**.  
- Analysis of content durations shows how **movie lengths evolved over the years**.
